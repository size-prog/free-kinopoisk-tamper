Userscript для Tampermonkey / Greasemonkey, добавляющий кнопку  
«Смотреть бесплатно» на страницах фильмов и сериалов Кинопоиска.

---

## Установка

### 1. Установить менеджер userscript’ов

Tampermonkey:  
https://www.tampermonkey.net/

Greasemonkey (Firefox):  
https://www.greasespot.net/

Установите расширение в браузер.

---

### 2. Установить скрипт

Перейдите по [Сюда](https://github.com/size-prog/free-kinopoisk-tamper/raw/refs/heads/main/main.user.js)

Откроется страница установки в Tampermonkey или Greasemonkey.  
Нажмите «Установить».

После установки дополнительных действий не требуется.
